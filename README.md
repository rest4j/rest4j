## Install

```xml
<dependency>
    <groupId>io.github.rest4j</groupId>
    <artifactId>rest4j-spring-boot-starter</artifactId>
    <version>0.1.4</version>
</dependency>
```

## Usage

```yaml
rest4j:
  default-exception-handler: true # 是否开启默认异常处理
```
